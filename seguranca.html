Segurança

Seu PC protegido como nunca

Dicas reais que bloqueiam 99% das ameaças – sem complicação

Ativar Windows Defender em tempo real

Configurações → Atualização e Segurança → Windows Security → Proteção contra vírus → Sempre ligado

Verificar se tem vírus agora

Windows Security → Proteção contra vírus → Opções de verificação → Verificação completa

Usar senha forte + PIN

Configurações → Contas → Opções de entrada → PIN do Windows (mais rápido e seguro que senha)

Ativar BitLocker (criptografia)

Pesquise "BitLocker" → Ativar no disco C: (ninguém acessa sem sua senha)

Usar autenticação em duas etapas

Em todas as contas (Google, Microsoft, banco) → Ative 2FA com app (nunca SMS)

Nunca clique em links estranhos

90% dos vírus entram por e-mail ou WhatsApp. Duvide de tudo que parece "urgente"

Fazer backup toda semana

Configurações → Atualização e Segurança → Backup → Adicionar uma unidade (HD externo)

Desativar macros no Office

Word/Excel → Arquivo → Opções → Central de Confiabilidade → Desativar todas as macros

Usar gerenciador de senhas

Bitwarden (grátis) ou Microsoft Authenticator → Nunca repita senhas

Desativar execução automática de USB

Win + R → gpedit.msc → Configuração do Computador → Modelos Administrativos → Sistema → Removable Storage Access → Desativar tudo
