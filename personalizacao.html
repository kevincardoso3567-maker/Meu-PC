Personalização

Seu Windows com a sua cara

Tema escuro perfeito, barra transparente, ícones incríveis e muito mais

Tema escuro em tudo

Configurações → Personalização → Cores → Escuro + ativar no Windows e nos apps

Barra de tarefas transparente

Configurações → Personalização → Cores → Ativar "Efeitos de transparência"

Centralizar ícones da barra

Configurações → Personalização → Barra de tarefas → Alinhamento → Centralizado

Papel de parede animado

Baixe Lively Wallpaper (grátis) → transforma vídeos em papel de parede vivo

Ícones modernos (grátis)

DeviantArt → "Windows 11 icons pack" → clique direito na pasta → Propriedades → Personalizar

Cursor personalizado

Configurações → Dispositivos → Mouse → Opções adicionais → baixe pacotes no DeviantArt

Barra de tarefas pequena

Configurações → Personalização → Barra de tarefas → Usar botões pequenos

Relógio com segundos

Configurações → Hora e idioma → ativar "Mostrar segundos"

RoundedTB (cantos arredondados)

Baixe no Microsoft Store → deixa a barra de tarefas com cantos do Windows 11

Menu Iniciar transparente

TranslucentTB (grátis na Microsoft Store) → 100% transparente ou com blur

Esconder barra de tarefas automaticamente

Configurações → Personalização → Barra de tarefas → Ocultar automaticamente

Fonte personalizada no sistema

Baixe fonte .ttf → clique direito → Instalar para todos os usuários
